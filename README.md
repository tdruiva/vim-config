vim-config
==========

## Setup

Dependencies:
* gem install github-markup
* apt install xdotool


```
~ $ git clone https://github.com/tdruiva/vim-config.git ~/.vim
~ $ cd ~/.vim
~/.vim $ git submodule update --init
~/.vim $ mkdir swap undo autoload
curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim
```

That's it.
